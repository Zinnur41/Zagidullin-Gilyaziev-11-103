<p><b>Гапа́р Айти́евич Айти́ев</b> (<a href="/wiki/1912" class="mw-redirect" title="1912">1912</a>—<a href="/wiki/1984" class="mw-redirect" title="1984">1984</a>)&#160;— советский, киргизский <a href="/wiki/%D0%A5%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA" title="Художник">художник</a>-<a href="/wiki/%D0%96%D0%B8%D0%B2%D0%BE%D0%BF%D0%B8%D1%81%D0%B5%D1%86" class="mw-redirect" title="Живописец">живописец</a>, <a href="/wiki/%D0%A1%D1%86%D0%B5%D0%BD%D0%BE%D0%B3%D1%80%D0%B0%D1%84" class="mw-redirect" title="Сценограф">сценограф</a>, <a href="/wiki/%D0%9F%D0%B5%D0%B4%D0%B0%D0%B3%D0%BE%D0%B3" class="mw-redirect" title="Педагог">педагог</a>. <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BE%D0%B9_%D0%A1%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D0%B8%D1%81%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B3%D0%BE_%D0%A2%D1%80%D1%83%D0%B4%D0%B0" title="Герой Социалистического Труда">Герой Социалистического Труда</a> (<a href="/wiki/1982" class="mw-redirect" title="1982">1982</a>). <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA_%D0%A1%D0%A1%D0%A1%D0%A0" title="Народный художник СССР">Народный художник СССР</a> (<a href="/wiki/1971" class="mw-redirect" title="1971">1971</a>). Лауреат <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BF%D1%80%D0%B5%D0%BC%D0%B8%D1%8F_%D0%9A%D0%B8%D1%80%D0%B3%D0%B8%D0%B7%D1%81%D0%BA%D0%BE%D0%B9_%D0%A1%D0%A1%D0%A0_%D0%B8%D0%BC%D0%B5%D0%BD%D0%B8_%D0%A2%D0%BE%D0%BA%D1%82%D0%BE%D0%B3%D1%83%D0%BB%D0%B0_%D0%A1%D0%B0%D1%82%D1%8B%D0%BB%D0%B3%D0%B0%D0%BD%D0%BE%D0%B2%D0%B0" title="Государственная премия Киргизской ССР имени Токтогула Сатылганова">Государственной премии Киргизской ССР им. Т. Сатылганова</a>. Один из первых профессиональных киргизских художников.
</p>

<div class="mw-heading mw-heading2"><h2 id="Биография"><span id=".D0.91.D0.B8.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.8F"></span>Биография</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%90%D0%B9%D1%82%D0%B8%D0%B5%D0%B2,_%D0%93%D0%B0%D0%BF%D0%B0%D1%80_%D0%90%D0%B9%D1%82%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Биография»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%B9%D1%82%D0%B8%D0%B5%D0%B2,_%D0%93%D0%B0%D0%BF%D0%B0%D1%80_%D0%90%D0%B9%D1%82%D0%B8%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=1" title="Редактировать код раздела «Биография»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<figure class="mw-halign-right" typeof="mw:File/Thumb"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:Stamp_of_Kyrgyzstan_003a.jpg" class="mw-file-description"><img src="//upload.wikimedia.org/wikipedia/commons/thumb/5/5e/Stamp_of_Kyrgyzstan_003a.jpg/250px-Stamp_of_Kyrgyzstan_003a.jpg" decoding="async" width="250" height="179" class="mw-file-element" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/5e/Stamp_of_Kyrgyzstan_003a.jpg/375px-Stamp_of_Kyrgyzstan_003a.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/5e/Stamp_of_Kyrgyzstan_003a.jpg/500px-Stamp_of_Kyrgyzstan_003a.jpg 2x" data-file-width="628" data-file-height="450" /></a><figcaption>Картина Г. Айтиева на почтовой марке Киргизии. 1992&#160;г.</figcaption></figure>
<p>Родился <abbr title="по юлианскому календарю">15</abbr>&#160;<a href="/wiki/28_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="28 сентября">(28)&#160;сентября</a>&#160;<a href="/wiki/1912_%D0%B3%D0%BE%D0%B4" title="1912 год">1912 года</a> в селе 1-й Тулейкен (ныне&#160;— <a href="/wiki/%D0%9A%D0%B0%D1%80%D0%B0-%D0%A1%D1%83%D1%83%D1%81%D0%BA%D0%B8%D0%B9_%D1%80%D0%B0%D0%B9%D0%BE%D0%BD" title="Кара-Сууский район">Кара-Сууского района</a> <a href="/wiki/%D0%9E%D1%88%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Ошская область">Ошской области</a> <a href="/wiki/%D0%9A%D1%8B%D1%80%D0%B3%D1%8B%D0%B7%D1%81%D1%82%D0%B0%D0%BD" title="Кыргызстан">Кыргызстана</a>).
</p><p>В 1932&#160;году окончил Киргизский институт просвещения. В 1932—1934&#160;годах преподавал в педагогическом техникуме в <a href="/wiki/%D0%94%D0%B6%D0%B0%D0%BB%D0%B0%D0%BB-%D0%90%D0%B1%D0%B0%D0%B4" title="Джалал-Абад">Джалал-Абаде</a>.
</p><p>В <a href="/wiki/1935" class="mw-redirect" title="1935">1935</a>—<a href="/wiki/1938_%D0%B3%D0%BE%D0%B4" title="1938 год">1938&#160;годах</a> учился в <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%BE%D0%B5_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%BE%D0%B5_%D0%B0%D0%BA%D0%B0%D0%B4%D0%B5%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%BE%D0%B5_%D1%83%D1%87%D0%B8%D0%BB%D0%B8%D1%89%D0%B5_%D0%BF%D0%B0%D0%BC%D1%8F%D1%82%D0%B8_1905_%D0%B3%D0%BE%D0%B4%D0%B0" class="mw-redirect" title="Московское государственное академическое художественное училище памяти 1905 года">Московском областном художественном педагогическом училище изобразительных искусств памяти восстания 1905&#160;года</a> у <a href="/wiki/%D0%9A%D1%80%D1%8B%D0%BC%D0%BE%D0%B2,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%9F%D0%B5%D1%82%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Крымов, Николай Петрович">Н.&#160;П.&#160;Крымова</a>.
</p><p>Участник <a href="/wiki/%D0%92%D0%9E%D0%92" class="mw-redirect" title="ВОВ">войны</a>. С декабря 1942&#160;года воевал на <a href="/wiki/%D0%AE%D0%B3%D0%BE-%D0%97%D0%B0%D0%BF%D0%B0%D0%B4%D0%BD%D1%8B%D0%B9_%D1%84%D1%80%D0%BE%D0%BD%D1%82_(%D0%92%D0%B5%D0%BB%D0%B8%D0%BA%D0%B0%D1%8F_%D0%9E%D1%82%D0%B5%D1%87%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%B2%D0%BE%D0%B9%D0%BD%D0%B0)" title="Юго-Западный фронт (Великая Отечественная война)">Юго-Западном фронте</a> в должности заместителя командира пулемётной роты по политчасти 2-го отдельного стрелкового батальона 94-й отдельной стрелковой бригады. Участвовал в <a href="/wiki/%D0%9E%D0%BF%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F_%C2%AB%D0%A3%D1%80%D0%B0%D0%BD%C2%BB" title="Операция «Уран»">Сталинградской битве</a>. 6 января 1943&#160;года был ранен. Гвардии старший лейтенант в отставке.
</p><p>Автор поэтичных, полных эпического спокойствия пейзажей «Колхозный двор» (<a href="/wiki/1946" class="mw-redirect" title="1946">1946</a>, Киргизский музей изобразительных искусств, Фрунзе), «Полдень на Иссык-Куле» (<a href="/wiki/1954" class="mw-redirect" title="1954">1954</a>, Третьяковская галерея), «Окрестности Андижана» (<a href="/wiki/1967" class="mw-redirect" title="1967">1967</a>, Киргизский музей изобразительных искусств, Фрунзе), ряда портретов, жанровых полотен и монументальных росписей, графических работ<sup id="cite_ref-1" class="reference"><a href="#cite_note-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup>.
</p><p>В 1938—1940&#160;годах&#160;— художник-постановщик Киргизского музыкально-драматического театра. Оформил спектакли: «Алтын-кыз» («Золотая девушка») <a href="/wiki/%D0%92%D0%BB%D0%B0%D1%81%D0%BE%D0%B2,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87_(%D0%BA%D0%BE%D0%BC%D0%BF%D0%BE%D0%B7%D0%B8%D1%82%D0%BE%D1%80)" title="Власов, Владимир Александрович (композитор)">В.&#160;Власова</a>, <a href="/wiki/%D0%A4%D0%B5%D1%80%D0%B5,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%93%D0%B5%D0%BE%D1%80%D0%B3%D0%B8%D0%B5%D0%B2%D0%B8%D1%87" title="Фере, Владимир Георгиевич">В.&#160;Фере</a> и <a href="/wiki/%D0%9C%D0%B0%D0%BB%D0%B4%D1%8B%D0%B1%D0%B0%D0%B5%D0%B2,_%D0%90%D0%B1%D0%B4%D1%8B%D0%BB%D0%B0%D1%81_%D0%9C%D0%B0%D0%BB%D0%B4%D1%8B%D0%B1%D0%B0%D0%B5%D0%B2%D0%B8%D1%87" title="Малдыбаев, Абдылас Малдыбаевич">А. Малдыбаева</a> (1939), «Токтогул» А.&#160;М.&#160;Верника (1940) и др.
</p><p>С <a href="/wiki/1977_%D0%B3%D0%BE%D0%B4" title="1977 год">1977&#160;года</a> руководил творческой мастерской живописи АХ СССР во <a href="/wiki/%D0%91%D0%B8%D1%88%D0%BA%D0%B5%D0%BA" title="Бишкек">Фрунзе</a>.
</p><p>Автор книги «Изобразительное искусство Киргизии» (Фрунзе, 1964).
</p><p>Член-корреспондент <a href="/wiki/%D0%90%D0%A5_%D0%A1%D0%A1%D0%A1%D0%A0" class="mw-redirect" title="АХ СССР">Академии художеств СССР</a> (<a href="/wiki/1973" class="mw-redirect" title="1973">1973</a>). В <a href="/wiki/1940" class="mw-redirect" title="1940">1940</a>—<a href="/wiki/1941" class="mw-redirect" title="1941">1941</a>, <a href="/wiki/1944" class="mw-redirect" title="1944">1944</a>—<a href="/wiki/1951" class="mw-redirect" title="1951">1951</a> и <a href="/wiki/1952" class="mw-redirect" title="1952">1952</a>—<a href="/wiki/1977_%D0%B3%D0%BE%D0%B4" title="1977 год">1977&#160;годах</a>&#160;— председатель правления Союза художников Киргизии. Член <a href="/wiki/%D0%A1%D0%BE%D1%8E%D0%B7_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2_%D0%A1%D0%A1%D0%A1%D0%A0" title="Союз художников СССР">Союза художников СССР</a>.
</p><p>Член <a href="/wiki/%D0%92%D0%9A%D0%9F(%D0%B1)" class="mw-redirect" title="ВКП(б)">ВКП(б)</a> с <a href="/wiki/1939_%D0%B3%D0%BE%D0%B4" title="1939 год">1939&#160;года</a>. Депутат <a href="/wiki/%D0%92%D0%B5%D1%80%D1%85%D0%BE%D0%B2%D0%BD%D1%8B%D0%B9_%D0%A1%D0%BE%D0%B2%D0%B5%D1%82_%D0%9A%D0%B8%D1%80%D0%B3%D0%B8%D0%B7%D1%81%D0%BA%D0%BE%D0%B9_%D0%A1%D0%A1%D0%A0" title="Верховный Совет Киргизской ССР">Верховного Совета Киргизской ССР</a> 5—9 созывов. Заместитель председателя Верховного Совета Киргизской ССР (с 1971).
</p><p>Скончался 16 декабря 1984&#160;года в Бишкеке. Похоронен на <a href="/wiki/%D0%90%D0%BB%D0%B0-%D0%90%D1%80%D1%87%D0%B8%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BB%D0%B0%D0%B4%D0%B1%D0%B8%D1%89%D0%B5" title="Ала-Арчинское кладбище">Ала-Арчинском кладбище</a>.
</p>
