<p><b>Элгу́джа Дави́дович Амашуке́ли</b> (<a href="/wiki/%D0%93%D1%80%D1%83%D0%B7%D0%B8%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D1%8F%D0%B7%D1%8B%D0%BA" title="Грузинский язык">груз.</a> <span lang="ka">ელგუჯა დავითის ძე ამაშუკელი</span>; <a href="/wiki/1928" class="mw-redirect" title="1928">1928</a>—<a href="/wiki/2002" class="mw-redirect" title="2002">2002</a><sup id="cite_ref-5" class="reference"><a href="#cite_note-5"><span class="cite-bracket">&#91;</span>5<span class="cite-bracket">&#93;</span></a></sup>)&#160;— <a href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" class="mw-redirect" title="СССР">советский</a>, <a href="/wiki/%D0%93%D1%80%D1%83%D0%B7%D0%B8%D1%8F" title="Грузия">грузинский</a> <a href="/wiki/%D0%A1%D0%BA%D1%83%D0%BB%D1%8C%D0%BF%D1%82%D0%BE%D1%80" class="mw-redirect" title="Скульптор">скульптор</a>, <a href="/wiki/%D0%A5%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA" title="Художник">художник</a>. <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA_%D0%A1%D0%A1%D0%A1%D0%A0" title="Народный художник СССР">Народный художник СССР</a> (<a href="/wiki/1988" class="mw-redirect" title="1988">1988</a>). Лауреат <a href="/wiki/%D0%93%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D0%BF%D1%80%D0%B5%D0%BC%D0%B8%D1%8F_%D0%A1%D0%A1%D0%A1%D0%A0" title="Государственная премия СССР">Государственной премии СССР</a> (<a href="/wiki/1982" class="mw-redirect" title="1982">1982</a>).
</p>
<div class="mw-heading mw-heading2"><h2 id="Биография"><span id=".D0.91.D0.B8.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.8F"></span>Биография</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%90%D0%BC%D0%B0%D1%88%D1%83%D0%BA%D0%B5%D0%BB%D0%B8,_%D0%AD%D0%BB%D0%B3%D1%83%D0%B4%D0%B6%D0%B0_%D0%94%D0%B0%D0%B2%D0%B8%D0%B4%D0%BE%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Биография»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BC%D0%B0%D1%88%D1%83%D0%BA%D0%B5%D0%BB%D0%B8,_%D0%AD%D0%BB%D0%B3%D1%83%D0%B4%D0%B6%D0%B0_%D0%94%D0%B0%D0%B2%D0%B8%D0%B4%D0%BE%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=1" title="Редактировать код раздела «Биография»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Элгуджа Амашукели родился <a href="/wiki/22_%D0%B0%D0%BF%D1%80%D0%B5%D0%BB%D1%8F" title="22 апреля">22 апреля</a> <a href="/wiki/1928_%D0%B3%D0%BE%D0%B4" title="1928 год">1928 года</a> в Тифлисе (ныне <a href="/wiki/%D0%A2%D0%B1%D0%B8%D0%BB%D0%B8%D1%81%D0%B8" title="Тбилиси">Тбилиси</a>, <a href="/wiki/%D0%93%D1%80%D1%83%D0%B7%D0%B8%D1%8F" title="Грузия">Грузия</a>).
</p><p>Учился в 1947—1954 годах в <a href="/wiki/%D0%A2%D0%B1%D0%B8%D0%BB%D0%B8%D1%81%D1%81%D0%BA%D0%B0%D1%8F_%D0%B0%D0%BA%D0%B0%D0%B4%D0%B5%D0%BC%D0%B8%D1%8F_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%B5%D1%81%D1%82%D0%B2" class="mw-redirect" title="Тбилисская академия художеств">Тбилисской академии художеств</a> у <a href="/wiki/%D0%9A%D0%B0%D0%BD%D0%B4%D0%B5%D0%BB%D0%B0%D0%BA%D0%B8,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%9F%D0%BE%D1%80%D1%84%D0%B8%D1%80%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Канделаки, Николай Порфирьевич">Н. П. Канделаки</a>. После окончания академии продолжил обучение в аспирантуре (1958—1959), где работал над проектом памятника <a href="/wiki/%D0%93%D0%BE%D1%80%D0%B3%D0%B0%D1%81%D0%B0%D0%BB%D0%B8" class="mw-redirect" title="Горгасали">Вахтангу Горгасали</a>.
</p><p>Автор скульптуры «<a href="/wiki/%D0%9C%D0%B0%D1%82%D1%8C_%D0%93%D1%80%D1%83%D0%B7%D0%B8%D0%B8" class="mw-redirect" title="Мать Грузии">Мать Грузии</a>», барельефов на станции метро «<a href="/wiki/%D0%A0%D1%83%D1%81%D1%82%D0%B0%D0%B2%D0%B5%D0%BB%D0%B8_(%D1%81%D1%82%D0%B0%D0%BD%D1%86%D0%B8%D1%8F_%D0%BC%D0%B5%D1%82%D1%80%D0%BE)" title="Руставели (станция метро)">Площадь Руставели</a>» (1966), <a href="/wiki/%D0%9F%D0%B0%D0%BC%D1%8F%D1%82%D0%BD%D0%B8%D0%BA_%D0%92%D0%B0%D1%85%D1%82%D0%B0%D0%BD%D0%B3%D1%83_%D0%93%D0%BE%D1%80%D0%B3%D0%B0%D1%81%D0%B0%D0%BB%D0%B8" title="Памятник Вахтангу Горгасали">памятника Вахтангу Горгасали</a> (1967) в <a href="/wiki/%D0%A2%D0%B1%D0%B8%D0%BB%D0%B8%D1%81%D0%B8" title="Тбилиси">Тбилиси</a>, монумента Победы в <a href="/wiki/%D0%93%D0%BE%D1%80%D0%B8" title="Гори">Гори</a> (1979), мемориала Славы героям-морякам в <a href="/wiki/%D0%9F%D0%BE%D1%82%D0%B8" title="Поти">Поти</a> (1979)<sup id="cite_ref-bse-ama_6-0" class="reference"><a href="#cite_note-bse-ama-6"><span class="cite-bracket">&#91;</span>6<span class="cite-bracket">&#93;</span></a></sup><sup id="cite_ref-7" class="reference"><a href="#cite_note-7"><span class="cite-bracket">&#91;</span>7<span class="cite-bracket">&#93;</span></a></sup>.
</p><p>Иллюстрировал и оформил книги для издательств «Сабчота Сакартвело», «Накадули»: «Песня о Нибелунгах» (1960), «Сочинения» Важа-Пшавела (1962—1964, 5 т.). Автор серии картин «Фантазия леса» (1962—1964), ряда <a href="/wiki/%D0%9C%D0%B5%D0%B4%D0%B0%D0%BB%D1%8C%D0%BE%D0%BD" title="Медальон">медальонов</a>.
</p><p>С <a href="/wiki/1996_%D0%B3%D0%BE%D0%B4" title="1996 год">1996 года</a> — член-корреспондент отделения языка и литературы <a href="/wiki/%D0%90%D0%BA%D0%B0%D0%B4%D0%B5%D0%BC%D0%B8%D1%8F_%D0%BD%D0%B0%D1%83%D0%BA_%D0%93%D1%80%D1%83%D0%B7%D0%B8%D0%B8" class="mw-redirect" title="Академия наук Грузии">Академии наук Грузии</a>.
</p><p>Председатель правления Союза художников Грузии (1981–1996). Секретарь <a href="/wiki/%D0%A1%D0%BE%D1%8E%D0%B7_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2_%D0%A1%D0%A1%D0%A1%D0%A0" title="Союз художников СССР">Союза художников СССР</a> (1984—1989).
</p><p>Член-корреспондент АХ СССР (1988).
