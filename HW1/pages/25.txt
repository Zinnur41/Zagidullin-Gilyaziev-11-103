<p><b>Самуил Яковлевич Адливанкин</b> (<abbr title="по юлианскому календарю">7</abbr>&#160;<a href="/wiki/19_%D0%B8%D1%8E%D0%BB%D1%8F" title="19 июля">&#91;19&#93;&#160;июля</a>&#160;<a href="/wiki/1897_%D0%B3%D0%BE%D0%B4" title="1897 год">1897</a>, <a href="/wiki/%D0%A2%D0%B0%D1%82%D0%B0%D1%80%D1%81%D0%BA_(%D0%A1%D0%BC%D0%BE%D0%BB%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C)" title="Татарск (Смоленская область)">Татарск</a>, <a href="/wiki/%D0%9C%D0%BE%D0%B3%D0%B8%D0%BB%D1%91%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D0%B3%D1%83%D0%B1%D0%B5%D1%80%D0%BD%D0%B8%D1%8F" title="Могилёвская губерния">Могилёвская губерния</a>&#160;— <a href="/wiki/7_%D0%BC%D0%B0%D1%80%D1%82%D0%B0" title="7 марта">7 марта</a> <a href="/wiki/1966_%D0%B3%D0%BE%D0%B4" title="1966 год">1966&#160;года</a>, <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москва</a>)&#160;— советский <a href="/wiki/%D0%A5%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA" title="Художник">художник</a>.

<div class="mw-heading mw-heading2"><h2 id="Биография"><span id=".D0.91.D0.B8.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.8F"></span>Биография</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%90%D0%B4%D0%BB%D0%B8%D0%B2%D0%B0%D0%BD%D0%BA%D0%B8%D0%BD,_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B2%D0%BB%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Биография»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%B4%D0%BB%D0%B8%D0%B2%D0%B0%D0%BD%D0%BA%D0%B8%D0%BD,_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B2%D0%BB%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=1" title="Редактировать код раздела «Биография»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Родился в местечке Татарск Могилёвской губернии (ныне&#160;— <a href="/wiki/%D0%A1%D0%BC%D0%BE%D0%BB%D0%B5%D0%BD%D1%81%D0%BA%D0%B0%D1%8F_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C" title="Смоленская область">Смоленская область</a>), в семье <a href="/wiki/%D0%A4%D0%B0%D1%80%D0%BC%D0%B0%D1%86%D0%B5%D0%B2%D1%82" class="mw-redirect" title="Фармацевт">фармацевта</a> Якова Ароновича Адливанкина и домохозяйки Сары Моисеевны Адливанкиной. Отец погиб в 1914 году. Жил и работал в Татарске, <a href="/wiki/%D0%9E%D0%B4%D0%B5%D1%81%D1%81%D0%B0" title="Одесса">Одессе</a>, <a href="/wiki/%D0%A1%D0%B0%D0%BC%D0%B0%D1%80%D0%B0" title="Самара">Самаре</a>, <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москве</a>.
</p><p>1912—1917 годы&#160;— обучение в <a href="/wiki/%D0%9E%D0%B4%D0%B5%D1%81%D1%81%D0%BA%D0%BE%D0%B5_%D1%82%D0%B5%D0%B0%D1%82%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE-%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%BE%D0%B5_%D1%83%D1%87%D0%B8%D0%BB%D0%B8%D1%89%D0%B5" title="Одесское театрально-художественное училище">Одесском художественном училище</a>, в классе у <a href="/wiki/%D0%9A%D0%BE%D1%81%D1%82%D0%B0%D0%BD%D0%B4%D0%B8,_%D0%9A%D0%B8%D1%80%D0%B8%D0%B0%D0%BA_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B0%D0%BD%D1%82%D0%B8%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Костанди, Кириак Константинович">К.&#160;К.&#160;Костанди</a>, по выходу получил диплом учителя рисования и <a href="/wiki/%D0%A7%D0%B5%D1%80%D1%87%D0%B5%D0%BD%D0%B8%D0%B5" class="mw-redirect" title="Черчение">черчения</a>.
</p><p>1918&#160;— учился у <a href="/wiki/%D0%A2%D0%B0%D1%82%D0%BB%D0%B8%D0%BD,_%D0%92%D0%BB%D0%B0%D0%B4%D0%B8%D0%BC%D0%B8%D1%80_%D0%95%D0%B2%D0%B3%D1%80%D0%B0%D1%84%D0%BE%D0%B2%D0%B8%D1%87" title="Татлин, Владимир Евграфович">В.&#160;Е.&#160;Татлина</a> в <a href="/w/index.php?title=%D0%A1%D0%B2%D0%BE%D0%B1%D0%BE%D0%B4%D0%BD%D1%8B%D0%B5_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B5_%D0%BC%D0%B0%D1%81%D1%82%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D0%B5&amp;action=edit&amp;redlink=1" class="new" title="Свободные художественные мастерские (страница отсутствует)">Свободных художественных мастерских</a> в Москве.
</p><p>1919&#160;— по поручению Отдела ИЗО <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%BC%D0%B8%D1%81%D1%81%D0%B0%D1%80%D0%B8%D0%B0%D1%82_%D0%BF%D1%80%D0%BE%D1%81%D0%B2%D0%B5%D1%89%D0%B5%D0%BD%D0%B8%D1%8F_%D0%A0%D0%A1%D0%A4%D0%A1%D0%A0" title="Народный комиссариат просвещения РСФСР">Наркомпроса</a> занимался организацией «Свомас» (Свободных мастерских) в <a href="/wiki/%D0%A1%D0%B0%D0%BC%D0%B0%D1%80%D0%B0" title="Самара">Самаре</a>, там же назначен руководителем художественной секции губернского комитета по масскультуре, отвечал за развитие художественного образования и программ по искусству для детей.
</p><p>1921&#160;— в составе членов-учредителей художественного объединения <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%BE%D0%B5_%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%BE_%D0%B6%D0%B8%D0%B2%D0%BE%D0%BF%D0%B8%D1%81%D1%86%D0%B5%D0%B2" title="Новое общество живописцев">Нового общества живописцев</a> (по 1922).
</p><p>1927&#160;— работал над фильмом «<a href="/wiki/%D0%A7%D0%B0%D1%88%D0%BA%D0%B0_%D1%87%D0%B0%D1%8F" title="Чашка чая">Чашка чая</a>» на «<a href="/wiki/%D0%9C%D0%B5%D0%B6%D1%80%D0%B0%D0%B1%D0%BF%D0%BE%D0%BC%D1%84%D0%B8%D0%BB%D1%8C%D0%BC" title="Межрабпомфильм">Межрабпомфильме</a>».
</p><p>1932&#160;— член общества «<a href="/w/index.php?title=%D0%98%D0%B7%D0%BE%D0%B1%D1%80%D0%B8%D0%B3%D0%B0%D0%B4%D0%B0&amp;action=edit&amp;redlink=1" class="new" title="Изобригада (страница отсутствует)">Изобригада</a>».
</p><p>1938—1939&#160;— создавал росписи для <a href="/w/index.php?title=%D0%9F%D0%B0%D0%B2%D0%B8%D0%BB%D1%8C%D0%BE%D0%BD_%C2%AB%D0%94%D0%B0%D0%BB%D1%8C%D0%BD%D0%B8%D0%B9_%D0%92%D0%BE%D1%81%D1%82%D0%BE%D0%BA%C2%BB&amp;action=edit&amp;redlink=1" class="new" title="Павильон «Дальний Восток» (страница отсутствует)">павильона «Дальний Восток»</a> на <a href="/wiki/%D0%92%D0%A1%D0%A5%D0%92" class="mw-redirect" title="ВСХВ">Всесоюзной сельскохозяйственной выставке</a> в Москве (в монументальном стиле).
</p><p>Урна с прахом захоронена в колумбарии <a href="/wiki/%D0%9D%D0%BE%D0%B2%D0%BE%D0%B5_%D0%94%D0%BE%D0%BD%D1%81%D0%BA%D0%BE%D0%B5_%D0%BA%D0%BB%D0%B0%D0%B4%D0%B1%D0%B8%D1%89%D0%B5" title="Новое Донское кладбище">Донского кладбища</a>.
</p>
<div class="mw-heading mw-heading2"><h2 id="Семья"><span id=".D0.A1.D0.B5.D0.BC.D1.8C.D1.8F"></span>Семья</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%90%D0%B4%D0%BB%D0%B8%D0%B2%D0%B0%D0%BD%D0%BA%D0%B8%D0%BD,_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B2%D0%BB%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Семья»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%B4%D0%BB%D0%B8%D0%B2%D0%B0%D0%BD%D0%BA%D0%B8%D0%BD,_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B2%D0%BB%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=2" title="Редактировать код раздела «Семья»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<ul><li>Жена (с 1918&#160;года)&#160;— Ханна Абрамовна Адливанкина (урождённая Низель, 1893—1959), родом из Одессы.
<ul><li>Сын&#160;— Михаил Самуилович Адливанкин (7 июня 1922&#160;— 25 января 1944), лейтенант, погиб на фронте в Псковской области.</li>
<li>Дочь&#160;— Ева Самуиловна Адливанкина (23 февраля 1920&#160;— 2 ноября 1990), художница по ткани</li></ul></li></ul>
<div class="mw-heading mw-heading2"><h2 id="Творчество"><span id=".D0.A2.D0.B2.D0.BE.D1.80.D1.87.D0.B5.D1.81.D1.82.D0.B2.D0.BE"></span>Творчество</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%90%D0%B4%D0%BB%D0%B8%D0%B2%D0%B0%D0%BD%D0%BA%D0%B8%D0%BD,_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B2%D0%BB%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=3" title="Редактировать раздел «Творчество»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%B4%D0%BB%D0%B8%D0%B2%D0%B0%D0%BD%D0%BA%D0%B8%D0%BD,_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B2%D0%BB%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=3" title="Редактировать код раздела «Творчество»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Живописец, монументалист, график, входил в «левое» крыло советского искусства 1920—1930 годов. Работал с плакатом, писал портреты, жанровые картины, пейзажи. Стилистика работ близка лубку, примитиву, работал в духе <a href="/wiki/%D0%A0%D0%BE%D0%B4%D1%87%D0%B5%D0%BD%D0%BA%D0%BE,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Родченко, Александр Михайлович">А.&#160;М.&#160;Родченко</a>, <a href="/wiki/%D0%9F%D0%BE%D0%BF%D0%BE%D0%B2%D0%B0,_%D0%9B%D1%8E%D0%B1%D0%BE%D0%B2%D1%8C_%D0%A1%D0%B5%D1%80%D0%B3%D0%B5%D0%B5%D0%B2%D0%BD%D0%B0" title="Попова, Любовь Сергеевна">Л.&#160;С.&#160;Поповой</a>, делал <a href="/wiki/%D0%A1%D1%82%D0%B0%D0%BD%D0%BA%D0%BE%D0%B2%D0%B0%D1%8F_%D0%B6%D0%B8%D0%B2%D0%BE%D0%BF%D0%B8%D1%81%D1%8C" title="Станковая живопись">станковые</a> работы в <a href="/wiki/%D0%93%D1%80%D0%BE%D1%82%D0%B5%D1%81%D0%BA" title="Гротеск">гротесковом</a> стиле. Художественную ценность представляют в основном работы до 1925 года.
</p>
<div class="mw-heading mw-heading3"><h3 id="Выставки"><span id=".D0.92.D1.8B.D1.81.D1.82.D0.B0.D0.B2.D0.BA.D0.B8"></span>Выставки</h3><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%90%D0%B4%D0%BB%D0%B8%D0%B2%D0%B0%D0%BD%D0%BA%D0%B8%D0%BD,_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B2%D0%BB%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=4" title="Редактировать раздел «Выставки»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%B4%D0%BB%D0%B8%D0%B2%D0%B0%D0%BD%D0%BA%D0%B8%D0%BD,_%D0%A1%D0%B0%D0%BC%D1%83%D0%B8%D0%BB_%D0%AF%D0%BA%D0%BE%D0%B2%D0%BB%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=4" title="Редактировать код раздела «Выставки»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
