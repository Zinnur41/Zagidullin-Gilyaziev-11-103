<p><b>Никола́й Ю́рьевич Ано́хин</b> (родился <a href="/wiki/6_%D1%81%D0%B5%D0%BD%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="6 сентября">6 сентября</a> <a href="/wiki/1966_%D0%B3%D0%BE%D0%B4" title="1966 год">1956&#160;года</a>, <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москва</a>, <a href="/wiki/%D0%A1%D0%BE%D1%8E%D0%B7_%D0%A1%D0%BE%D0%B2%D0%B5%D1%82%D1%81%D0%BA%D0%B8%D1%85_%D0%A1%D0%BE%D1%86%D0%B8%D0%B0%D0%BB%D0%B8%D1%81%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D1%85_%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA" title="Союз Советских Социалистических Республик">СССР</a>)&#160;— советский и российский <a href="/wiki/%D0%A5%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA" title="Художник">художник</a>, живописец.

<div class="mw-heading mw-heading2"><h2 id="Биография"><span id=".D0.91.D0.B8.D0.BE.D0.B3.D1.80.D0.B0.D1.84.D0.B8.D1.8F"></span>Биография</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%90%D0%BD%D0%BE%D1%85%D0%B8%D0%BD,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%AE%D1%80%D1%8C%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=1" title="Редактировать раздел «Биография»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BD%D0%BE%D1%85%D0%B8%D0%BD,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%AE%D1%80%D1%8C%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=1" title="Редактировать код раздела «Биография»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Отец — Анохин Юрий Николаевич (<a href="/wiki/1926" class="mw-redirect" title="1926">1926</a>—<a href="/wiki/1977" class="mw-redirect" title="1977">1977</a>), заслуженный художник РСФСР.
</p><p>Николай Анохин окончил <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D0%B3%D0%BE%D1%81%D1%83%D0%B4%D0%B0%D1%80%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%B0%D0%BA%D0%B0%D0%B4%D0%B5%D0%BC%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%B8%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%B8%D0%BC%D0%B5%D0%BD%D0%B8_%D0%92._%D0%98._%D0%A1%D1%83%D1%80%D0%B8%D0%BA%D0%BE%D0%B2%D0%B0" class="mw-redirect" title="Московский государственный академический художественный институт имени В. И. Сурикова">Московский государственный академический художественный институт имени В. И. Сурикова</a>. Дипломная работа&#160;— историческая картина «Воззвание Минина».
</p><p>В <a href="/wiki/1988" class="mw-redirect" title="1988">1988</a>—<a href="/wiki/1992" class="mw-redirect" title="1992">1992</a> прошёл стажировку в Творческой мастерской живописи <a href="/wiki/%D0%90%D0%A5_%D0%A1%D0%A1%D0%A1%D0%A0" class="mw-redirect" title="АХ СССР">Академии художеств СССР</a>.
</p>
<div class="mw-heading mw-heading2"><h2 id="Творческая_деятельность"><span id=".D0.A2.D0.B2.D0.BE.D1.80.D1.87.D0.B5.D1.81.D0.BA.D0.B0.D1.8F_.D0.B4.D0.B5.D1.8F.D1.82.D0.B5.D0.BB.D1.8C.D0.BD.D0.BE.D1.81.D1.82.D1.8C"></span>Творческая деятельность</h2><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=%D0%90%D0%BD%D0%BE%D1%85%D0%B8%D0%BD,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%AE%D1%80%D1%8C%D0%B5%D0%B2%D0%B8%D1%87&amp;veaction=edit&amp;section=2" title="Редактировать раздел «Творческая деятельность»" class="mw-editsection-visualeditor"><span>править</span></a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=%D0%90%D0%BD%D0%BE%D1%85%D0%B8%D0%BD,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%AE%D1%80%D1%8C%D0%B5%D0%B2%D0%B8%D1%87&amp;action=edit&amp;section=2" title="Редактировать код раздела «Творческая деятельность»"><span>править код</span></a><span class="mw-editsection-bracket">]</span></span></div>
<p>Творчество Николая Анохина представляет собой развитие <a href="/w/index.php?title=%D0%9A%D0%BB%D0%B0%D1%81%D1%81%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%BE%D0%B5_%D0%B8%D1%81%D0%BA%D1%83%D1%81%D1%81%D1%82%D0%B2%D0%BE&amp;action=edit&amp;redlink=1" class="new" title="Классическое искусство (страница отсутствует)">классического</a> направления русской живописи, включающего элементы <a href="/wiki/%D0%90%D0%BA%D0%B0%D0%B4%D0%B5%D0%BC%D0%B8%D0%B7%D0%BC" title="Академизм">академизма</a> и <a href="/wiki/%D0%A2%D1%80%D0%B0%D0%B4%D0%B8%D1%86%D0%B8%D0%BE%D0%BD%D0%B0%D0%BB%D0%B8%D0%B7%D0%BC" title="Традиционализм">традиционализма</a>.
</p><p>В <a href="/wiki/1987" class="mw-redirect" title="1987">1987</a>—<a href="/wiki/1993_%D0%B3%D0%BE%D0%B4" title="1993 год">1993&#160;годах</a> Н.&#160;Ю.&#160;Анохин работал с группой художников в северных областях России.
В <a href="/wiki/1993" class="mw-redirect" title="1993">1993</a>—<a href="/wiki/2000_%D0%B3%D0%BE%D0%B4" title="2000 год">2000&#160;годах</a> занимался воссозданием интерьеров, иконостаса и росписей Рождественского собора <a href="/wiki/%D0%91%D0%BE%D0%B3%D0%BE%D1%80%D0%BE%D0%B4%D0%B8%D1%86%D0%B5-%D0%A0%D0%BE%D0%B6%D0%B4%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9_%D0%BC%D0%BE%D0%BD%D0%B0%D1%81%D1%82%D1%8B%D1%80%D1%8C_(%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0)" class="mw-redirect" title="Богородице-Рождественский монастырь (Москва)">Богородице-Рождественского монастыря</a> в <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москве</a>.
</p>
<figure class="mw-halign-left" typeof="mw:File/Thumb"><a href="/wiki/%D0%A4%D0%B0%D0%B9%D0%BB:V_starom_dome_Rakitinyh_1998_by_Nikolay_Anokhin.jpg" class="mw-file-description"><img src="//upload.wikimedia.org/wikipedia/commons/thumb/a/ae/V_starom_dome_Rakitinyh_1998_by_Nikolay_Anokhin.jpg/320px-V_starom_dome_Rakitinyh_1998_by_Nikolay_Anokhin.jpg" decoding="async" width="320" height="236" class="mw-file-element" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/ae/V_starom_dome_Rakitinyh_1998_by_Nikolay_Anokhin.jpg/480px-V_starom_dome_Rakitinyh_1998_by_Nikolay_Anokhin.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/ae/V_starom_dome_Rakitinyh_1998_by_Nikolay_Anokhin.jpg/640px-V_starom_dome_Rakitinyh_1998_by_Nikolay_Anokhin.jpg 2x" data-file-width="705" data-file-height="520" /></a><figcaption>В старом доме Ракитиных. 1998</figcaption></figure>
<p>Является основателем творческого объединения «<a href="/wiki/%D0%A1%D0%BC%D0%B8%D1%80%D0%BD%D0%BE%D0%B2,_%D0%A1%D0%B5%D1%80%D0%B3%D0%B5%D0%B9_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87_(%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA,_1954)" title="Смирнов, Сергей Иванович (художник, 1954)">Русский Мир</a>» (<a href="/wiki/2002" class="mw-redirect" title="2002">2002</a>), в которое входят представители классического направления русской живописи, продолжающие и развивающие традиции <a href="/wiki/%D0%A1%D1%83%D1%80%D0%B8%D0%BA%D0%BE%D0%B2,_%D0%92%D0%B0%D1%81%D0%B8%D0%BB%D0%B8%D0%B9_%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2%D0%B8%D1%87" title="Суриков, Василий Иванович">В.&#160;И.&#160;Сурикова</a>, <a href="/wiki/%D0%92%D0%B0%D1%81%D0%BD%D0%B5%D1%86%D0%BE%D0%B2,_%D0%92%D0%B8%D0%BA%D1%82%D0%BE%D1%80_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Васнецов, Виктор Михайлович">В.&#160;М.&#160;Васнецова</a>, <a href="/wiki/%D0%98%D0%B2%D0%B0%D0%BD%D0%BE%D0%B2,_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80_%D0%90%D0%BD%D0%B4%D1%80%D0%B5%D0%B5%D0%B2%D0%B8%D1%87" title="Иванов, Александр Андреевич">А.&#160;А.&#160;Иванова</a>. Выставки творческого объединения «Русский мир» проходили в Москве <a href="/wiki/2002" class="mw-redirect" title="2002">2002</a>—<a href="/wiki/2004" class="mw-redirect" title="2004">2004</a>, <a href="/wiki/%D0%9C%D1%8E%D0%BD%D1%85%D0%B5%D0%BD" title="Мюнхен">Мюнхене</a> <a href="/wiki/2006" class="mw-redirect" title="2006">2006</a>, <a href="/wiki/%D0%92%D0%B0%D1%82%D0%B8%D0%BA%D0%B0%D0%BD" title="Ватикан">Ватикане</a> <a href="/wiki/2007" class="mw-redirect" title="2007">2007</a>.
</p><p>Работы Н.&#160;Ю.&#160;Анохина находятся в музейных собраниях и частных коллекциях <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">России</a>, <a href="/wiki/%D0%93%D0%B5%D1%80%D0%BC%D0%B0%D0%BD%D0%B8%D1%8F" title="Германия">Германии</a>, <a href="/wiki/%D0%90%D0%BD%D0%B3%D0%BB%D0%B8%D1%8F" title="Англия">Англии</a>, <a href="/wiki/%D0%AF%D0%BF%D0%BE%D0%BD%D0%B8%D1%8F" title="Япония">Японии</a>, <a href="/wiki/%D0%A1%D0%A8%D0%90" class="mw-redirect" title="США">США</a>, <a href="/wiki/%D0%A4%D1%80%D0%B0%D0%BD%D1%86%D0%B8%D1%8F" title="Франция">Франции</a>, <a href="/wiki/%D0%92%D0%B0%D1%82%D0%B8%D0%BA%D0%B0%D0%BD" title="Ватикан">Ватикана</a>.
</p>
