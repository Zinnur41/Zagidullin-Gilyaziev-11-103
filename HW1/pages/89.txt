<p><b>Аркадий Борисович Арсеньев</b> (<a href="/wiki/30_%D0%BE%D0%BA%D1%82%D1%8F%D0%B1%D1%80%D1%8F" title="30 октября">30 октября</a> <a href="/wiki/1939_%D0%B3%D0%BE%D0%B4" title="1939 год">1939&#160;года</a>, <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москва</a>, <a href="/wiki/%D0%A0%D0%A1%D0%A4%D0%A1%D0%A0" class="mw-redirect" title="РСФСР">РСФСР</a>, <a href="/wiki/%D0%A1%D0%A1%D0%A1%D0%A0" class="mw-redirect" title="СССР">СССР</a>&#160;— <a href="/wiki/2006_%D0%B3%D0%BE%D0%B4" title="2006 год">2006&#160;год</a>, <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%B2%D0%B0" title="Москва">Москва</a>, <a href="/wiki/%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D1%8F" title="Россия">Россия</a>)&#160;— советский и российский <a href="/wiki/%D0%A5%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA" title="Художник">художник</a>, график, живописец, плакатист, <a href="/wiki/%D0%9D%D0%B0%D1%80%D0%BE%D0%B4%D0%BD%D1%8B%D0%B9_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B9_%D0%A4%D0%B5%D0%B4%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D0%B8" title="Народный художник Российской Федерации">народный художник Российской Федерации</a> (2006).
</p><p>В 1958&#160;году&#160;— окончил <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B0%D1%8F_%D1%86%D0%B5%D0%BD%D1%82%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D0%B0%D1%8F_%D1%88%D0%BA%D0%BE%D0%BB%D0%B0" title="Московская центральная художественная школа">Московскую среднюю художественную школу</a>.
</p><p>В 1964&#160;году&#160;— окончил мастерскую плаката <a href="/wiki/%D0%9C%D0%BE%D1%81%D0%BA%D0%BE%D0%B2%D1%81%D0%BA%D0%B8%D0%B9_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9_%D0%B8%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%B8%D0%BC%D0%B5%D0%BD%D0%B8_%D0%92._%D0%98._%D0%A1%D1%83%D1%80%D0%B8%D0%BA%D0%BE%D0%B2%D0%B0" title="Московский художественный институт имени В. И. Сурикова">Московского государственного художественного института имени В.&#160;И.&#160;Сурикова</a>, где учился у <a href="/wiki/%D0%A7%D0%B5%D1%80%D0%B5%D0%BC%D0%BD%D1%8B%D1%85,_%D0%9C%D0%B8%D1%85%D0%B0%D0%B8%D0%BB_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Черемных, Михаил Михайлович">М.&#160;М.&#160;Черемных</a>, <a href="/wiki/%D0%9F%D0%BE%D0%BD%D0%BE%D0%BC%D0%B0%D1%80%D1%91%D0%B2,_%D0%9D%D0%B8%D0%BA%D0%BE%D0%BB%D0%B0%D0%B9_%D0%90%D1%84%D0%B0%D0%BD%D0%B0%D1%81%D1%8C%D0%B5%D0%B2%D0%B8%D1%87" title="Пономарёв, Николай Афанасьевич">Н.&#160;А.&#160;Пономарева</a>, <a href="/wiki/%D0%A1%D0%B0%D0%B2%D0%BE%D1%81%D1%82%D1%8E%D0%BA,_%D0%9E%D0%BB%D0%B5%D0%B3_%D0%9C%D0%B8%D1%85%D0%B0%D0%B9%D0%BB%D0%BE%D0%B2%D0%B8%D1%87" title="Савостюк, Олег Михайлович">О.&#160;М.&#160;Савостюка</a>, <a href="/wiki/%D0%A3%D1%81%D0%BF%D0%B5%D0%BD%D1%81%D0%BA%D0%B8%D0%B9,_%D0%91%D0%BE%D1%80%D0%B8%D1%81_%D0%90%D0%BB%D0%B5%D0%BA%D1%81%D0%B0%D0%BD%D0%B4%D1%80%D0%BE%D0%B2%D0%B8%D1%87" title="Успенский, Борис Александрович">Б.&#160;А.&#160;Успенского</a>.
</p><p>С 1964 участник московских, республиканских, всесоюзных и зарубежных выставок.
</p><p>С 1968&#160;года&#160;— член <a href="/wiki/%D0%A1%D0%BE%D1%8E%D0%B7_%D1%85%D1%83%D0%B4%D0%BE%D0%B6%D0%BD%D0%B8%D0%BA%D0%BE%D0%B2_%D0%A1%D0%A1%D0%A1%D0%A0" title="Союз художников СССР">Союза художников СССР</a>.
</p><p>С 1977&#160;года был заместителем председателя комиссии по плакату СХ РСФСР и председателем художественного совета творческо-производственной мастерской наглядной агитации ХФ РСФСР.
</p><p>Сотрудничал в издательстве «Плакат» (до 1974&#160;— в «Изобразительном искусстве»), постоянно работал в «Агитплакате».
</p><p>С 1989 по 2006&#160;годы&#160;— секретарь Союза художников России, заместитель (с 1979) и председатель плакатной секции МОСХа (1988—1989) и становился куратором по Дальнему Востоку.
</p><p>Основные произведения: «Берегите красоту и богатство нашей природы» (1967), «Родине&#160;— слава!» (1973), «Народ Чили победит!» (1974), «Пусть всегда будет солнце» (1975), «Для того, чтоб по рельсам проехать, надо прежде по шпалам пройти» (1976), «Слава строителям БАМа» (1976), «Честь и слава труду хлебороба» (1977), «Выше качество полевых работ» (1977), «Мы строим мир» (1978), «Вперед лети!» (1979).
</p><p>Тиражные плакаты находятся в коллекциях различных российских музеев.
</p>
